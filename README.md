# ED_DDX_EVALUATION

## 설치
pip install streamlit pandas

## 실행
streamlit run app.py

## 기능
- 좌측: 행 선택(Prev/Next), Core View(표), Optional Sections
- 우측: v3 패널
  - Physician DDX 작성(리스트)
  - Likert: Model DDX / Physician DDX
  - Likert: Current History / Past History
  - Comment
  - Save (행 단위 덮어쓰기)
  - Unreviewed 목록 + 점프
  - CSV 다운로드(중간 저장 가능)
